よくあるご質問
Q 休日はどのように決まりますか？
休日はご自身の都合で決めていただけます。
変更も自由です。
Q 自宅で開業していますがWワークは可能ですか？
勿論可能です！予約の入っていない日や時間帯に勤務していただいて大丈夫です。
スキマ時間をご活用ください。
Q 出張は初めてで不安です。
サロンに行けない、リラックスできる自宅で施術を受けたいという切実なお客様にご対応いただくためのサービスです。
セラピスト側にも安心していただけるようお客様も登録制にしております。
顧客情報の共有＆予約リクエスト時にセラピスト側で受注の可否もお選びいただけます。
Q 自宅待機は可能でしょうか？
可能です！自宅やカフェなどお好きな場所で待機していただけます。
Q 予約時間に遅れそうな場合はどうしたらいいですか？
お客様とはアプリ内でチャットやお電話が可能ですので遅れることが判明した時点で速やかにご連絡ください。
Q メニューの追加や変更があった場合はどうしたらいいですか？
決済前にメニューの変更なども可能です。
Q 身だしなみ・服装に決まりはありますか？
現時点で明確なルールはありませんが清潔感のある服装、髪型をお願いします。
お腹をマッサージしますので爪は短くしていただきストーンなどのネイルアートはお控えください。
Q どんな場所に呼ばれますか？
ご自宅やビジネスホテル、シティホテル、会社などになります。
Q お客様から急なキャンセルの依頼があった際の報酬はどうなりますか？
予約時間3時間前からのキャンセルは100％保証いたしますのでご安心ください。
Q 登録料や月額費用などはかかりますか？
登録や在籍にかかる費用は一切ありません。ご安心ください。
Q 報酬はどのように決まりますか？
報酬は60％～最大75％
出勤日数、予約数、在籍期間、お客様からの評価などを考慮し決定させていただきます。
施術料金はエリアに応じて目安もありますがご自身で設定いただくことも可能です。
Q 報酬はいつ支払われますか？
毎月1日～月末までの報酬を翌月25日に振り込みます。
